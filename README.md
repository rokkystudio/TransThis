# TransThis

TransThis — простое расширение Chrome/Edge для быстрого перевода выделенного текста через Google Translate.

## Возможности

- Показывает компактную кнопку с иконкой после выделения текста на странице.
- Открывает окно перевода после клика по кнопке.
- Использует автоматическое определение входящего языка.
- Позволяет выбрать язык перевода в окне.
- Показывает один перевод для фраз и предложений.
- Показывает перевод и словарные значения для одного слова, когда Google Translate возвращает словарь.
- Позволяет перетаскивать окно за шапку.
- Закрывает окно и кнопку при снятии выделения, новом выделении, клике по странице или Escape.
- Обрабатывает устаревший content script после перезагрузки расширения без падения `Extension context invalidated`.

## Установка в Edge

1. Открой `edge://extensions/`.
2. Включи режим разработчика.
3. Нажми `Load unpacked` / `Загрузить распакованное`.
4. Выбери папку `TransThis`.
5. Обнови уже открытые вкладки, на которых нужно использовать переводчик.

## Установка в Chrome

1. Открой `chrome://extensions/`.
2. Включи режим разработчика.
3. Нажми `Load unpacked` / `Загрузить распакованное`.
4. Выбери папку `TransThis`.
5. Обнови уже открытые вкладки, на которых нужно использовать переводчик.

## Ограничения

Расширение не запускается на внутренних страницах браузера, страницах магазина расширений и страницах, где браузер запрещает content scripts.

Для перевода используется публичный web-endpoint Google Translate без API-ключа. Это не официальный Google Cloud Translate API, поэтому формат ответа или доступность endpoint могут измениться.
