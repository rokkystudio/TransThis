# TransThis

TransThis — расширение Chrome/Edge для быстрого перевода выделенного текста через Google Translate.

## Возможности

- Показывает компактное облачко `Перевести` под выделением текста.
- Открывает окно перевода без шапки и без кнопки закрытия.
- Открывает окно перевода под выделением.
- Позволяет перетаскивать окно кликом по любой точке окна.
- Делает окно полупрозрачным во время перетаскивания.
- Показывает два селектора языков: язык исходного текста и язык перевода.
- Показывает для языка исходного текста пункт `Автоматически`.
- По умолчанию использует `Автоматически → Русский`.
- Запоминает выбранные языки в `chrome.storage.local` и применяет их в новых вкладках, окнах браузера и после перезапуска браузера.
- Для нескольких слов показывает точный перевод одной строкой.
- Для одного слова показывает перевод и словарные значения в стиле Google Translate без дублирования исходного выделенного текста.
- Закрывает окно и кнопку при снятии выделения, новом выделении, клике по странице или по Escape.
- Подавляет нативные элементы перевода Google на страницах Google, чтобы они не конфликтовали с интерфейсом расширения.
- Обрабатывает устаревший content script после перезагрузки расширения без падения `Extension context invalidated`.

## Установка в Edge

1. Открой `edge://extensions/`.
2. Включи режим разработчика.
3. Нажми `Load unpacked` / `Загрузить распакованное`.
4. Выбери папку `TransThis`.
5. Нажми `Reload` у расширения, если оно уже было установлено.
6. Обнови уже открытые вкладки, на которых нужно использовать переводчик.

## Установка в Chrome

1. Открой `chrome://extensions/`.
2. Включи режим разработчика.
3. Нажми `Load unpacked` / `Загрузить распакованное`.
4. Выбери папку `TransThis`.
5. Нажми `Reload` у расширения, если оно уже было установлено.
6. Обнови уже открытые вкладки, на которых нужно использовать переводчик.

## Ограничения

Расширение не запускается на внутренних страницах браузера, страницах магазина расширений и страницах, где браузер запрещает content scripts.

Для перевода используется публичный web-endpoint Google Translate без API-ключа. Это не официальный Google Cloud Translate API, поэтому формат ответа или доступность endpoint могут измениться.
